# 数据结构与算法 golang

## List of Algorithms

- linkedlist
  - array_singlylinkedlist： 数组存储单链表
  - singlylinkedlist： 链式存储单链表
  - doublylinkedlist： 双向链表
- queue
  - array_queue 数组存储队列
  - queue 链式存储队列
  - loop_queue 循环队列
- stack
  - array_stack 数组存储栈
  - stack 链式存储栈
- heap 大顶堆
- string  
  - KMP 算法
- tree
  - binary_tree 二叉树
  - bst 二叉查找树
  - huffman_tree 哈夫曼树
  - threaded_binary_tree 线索化二叉树
  - segment_tree 线段树
- 查找
  - 二分查找
  - 二分递归查找
  - 插值查找
  - 斐波那契数列查找
- 排序
  - 冒泡排序
  - 插入排序
  - 选择排序
  - 希尔排序
  - 堆排序
  - 归并排序
